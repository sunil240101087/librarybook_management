# Library Book Management System

This project implements a Library Book Management System using Python and in-memory data structures only (no database). The system was developed using a three-sprint Agile process with Git for version control and unit testing for verification.

## Features

- Add books with Book ID, Title, and Author
- Prevent duplicate Book IDs
- Borrow and return books
- Prevent borrowing of already borrowed books
- Generate a library status report
- Fully unit tested using Python unittest

## Technologies Used

- Python
- unittest (testing framework)
- Git & GitHub (version control)
