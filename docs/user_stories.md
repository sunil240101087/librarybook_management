# User Stories

US1: As a librarian, I want to add new books so that they are stored in the system.

US2: As a librarian, I want to prevent duplicate book IDs to avoid conflicts.

US3: As a librarian, I want to borrow a book so I can track lending.

US4: As a librarian, I want to return a book to mark it as available.

US5: As a librarian, I want to generate a report to view library status.
